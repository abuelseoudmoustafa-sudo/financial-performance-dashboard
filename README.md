# 📊 Financial Performance & Profitability Dashboard

## 📌 Project Overview

This Power BI dashboard analyzes financial performance by examining **sales, discounts, costs, and profitability** across products, segments, and countries.

The goal of this project is to provide business leaders with **clear insights into revenue drivers, profit margins, and pricing effectiveness** to support data-driven decision making.

## 📊 Key Metrics

- **Gross Sales:** $127.93M  
- **Discounts:** $9.21M  
- **Net Sales:** $118.73M  
- **Total Profit:** $16.89M  
- **Profit Margin:** 14.23%

---

## 🔍 Key Insights

### Product Performance
- **Paseo** generated the highest net sales (**$33M**).
- Followed by **VTT ($21M)** and **Velo ($18M)**.

### Profitability by Discount
- **Low and Medium discount levels** generated the largest profit contribution.
- **High discounts** significantly reduced overall profitability.

### Sales by Segment
- **Government** and **Small Business** segments generated the highest sales volumes.

### Profit by Country
Top profit-generating markets:

1. France — **$3.8M**
2. Germany — **$3.7M**
3. Canada — **$3.5M**
